# MVC-APPLICATION
BACKEND (PHP) DEVELOPER

This is an assignment to;

Create a laravel mvc application(front-end and back-end) and also make available API endpoints to
feed a mobile app version of the application (Mobile App not included).
The application will have the following features:  


1. Login and registration for patients and doctors (token based authentication for the API)

2. Appointment booking for patients and doctors

3. Dashboard to show activities with any chart library of choice

4. Responsive design

5. Email notification after registration
